# Heart Risk Prediction using Machine Learning
Explainable heart disease prediction using Random Forest and SHAP
This project predicts heart disease risk using clinical data.

Model: Random Forest  
Accuracy: 89%  
AUC Score: 0.95  

Explainability implemented using SHAP.

The system can:
- Predict heart disease risk
- Provide probability score
- Explain predictions using SHAP

This project demonstrates explainable AI in healthcare.
